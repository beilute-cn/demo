Windows Batch 脚本中的转义字符详解
需要转义的特殊字符列表

1. 核心特殊字符（必须转义）
字符	含义	转义方式	示例
^	转义字符本身	^^	echo ^^ → ^
&	命令分隔符	^&	echo A ^& B → A & B
`	`	管道	`^
<	输入重定向	^<	echo ^<file^> → <file>
>	输出重定向	^>	echo test ^> file → test > file
%	变量/参数	%% 或 ^%	echo 100%% → 100%
"	引号	^" 或 ""	echo ^"hello^" → "hello"
(	命令块开始	^(	echo ^(test^) → (test)
)	命令块结束	^)	echo ^(test^) → (test)
!	延迟展开变量	^^!	echo ^^!var^^! → !var!

2. 次要特殊字符（特定情况需要转义）
字符	含义	何时需要转义	转义方式
,	分隔符	在某些命令中	^,
;	分隔符	在某些命令中	^;
=	赋值	在某些命令中	^=
\	路径分隔符	通常不需要	-
/	选项前缀	通常不需要	-
-	选项前缀	通常不需要	-
*	通配符	在文件名中	^*
?	通配符	在文件名中	^?
[ ]	字符集	很少需要	^[ ^]

gcovr中的搜索路径列表含空格，使用双引号包括
在和续行符号紧挨着使用时，造成转义
同时格式化又会使下一行之前的空格无法保留
使用小括号包裹，使格式化无效，添加额外的空格

```shell{.line-numbers}
echo ^
"test"

不是 echo "test"
而是 echo ^"test"


```
